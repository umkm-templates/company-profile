# Company Profile UMKM (Next.js)

SEO-friendly company profile website built with **Next.js + Tailwind CSS**, designed specifically for **UMKM (Usaha Mikro, Kecil, dan Menengah)**.

This project focuses on:

- Fast loading
- Mobile-first design
- WhatsApp-based ordering
- Easy content customization
- Ready for Vercel deployment
